# ToDo List Application
This is a full-stack application that allows you to manage your to-do list. It consists of an Angular frontend for the user interface and a Spring Boot backend for handling the data and providing RESTful APIs. The application provides features such as creating, updating, and deleting tasks, as well as user authentication and authorization using Spring Security.

# Features
- User authentication and authorization
- Create, update, and delete tasks
- View tasks in a list
- Mark tasks as completed
- Filter tasks by status

# Technologies Used
- Angular
- HTML
- CSS
- TypeScript
- Spring Boot
- Spring Security
- RESTful APIs
- JPA/Hibernate

# Prerequisites
Node.js (v12 or higher)
Angular CLI (v12 or higher)
Java Development Kit (JDK) 8 or higher
Spring Boot (v2.4 or higher)

# Getting Started
Frontend
1. Clone the repository:
- git clone https://github.com/your-username/todo-list-app.git
2. Navigate to the frontend directory:
- cd todo-list-app/frontend
3. Install the dependencies:
- npm install
4. Start the development server:
- ng serve
5. Open your browser and visit http://localhost:4200 to access the application.

Backend
1. Navigate to the backend directory:
- cd ../backend
2. Import the project into your preferred IDE.
3. Build and run the application.
- The backend server will start on http://localhost:8080.

Usage
- Open the application in your browser.
- Sign up for a new account or log in if you already have one.
- Create, update, or delete tasks as needed.
- Use the filter options to view tasks based on their status.
- Mark tasks as completed when you finish them.
- Log out when you're done.

# Folder Structure
- frontend/ - Contains the Angular frontend application files.
- backend/ - Contains the Spring Boot backend application files.
- backend/src/main/java/com/example/todo/ - Contains the Java source code for the backend application.
- backend/src/main/resources/ - Contains the application configuration and properties files.
